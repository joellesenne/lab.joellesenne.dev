## Prerequisites

- npm >=7.13.0
- node >=16.0.0
- gatsby >=3.5.0

## Install

```sh
# gatsby-cli
npm install -g gatsby-cli

# Clone this repository
git clone  https://github.com/joellesenne/lab.joellesenne.dev.git

cd lab.joellesenne.dev

npm install

# OR
yarn install
```
