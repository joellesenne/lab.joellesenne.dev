## Empty project.

### First install dependencies:

```bash
~ $ cd <patch>

~ $ npm install -g parcel-bundler

~ $ npm install
```

### Building and running on localhost

```bash
# Start Parcel
~ $ parcel scr/index.html

# Build Parcel
~ $ parcel build scr/index.html
```

=> browse to [http://localhost:1234/](http://localhost:1234/)
